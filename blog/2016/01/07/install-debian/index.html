8.3 Keyboard

在 /etc/default/keyboard 文件中添加下面一行，交换 Ctrl 和 CapsLock 键：

XKBOPTIONS="ctrl:nocaps"

完整的 /etc/default/keyboard 文件如下:

XKBMODEL="pc105"

XKBVARIANT="" XKBOPTIONS="ctrl:nocaps"

BACKSPACE="guess"
