1 Install software via apt-get command

1.1 Change /etc/apt/sources.list

Take Ubuntu 16.04 as an example. First, edit the sources.list file with root privileges: 

# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-updates main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-backports main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-security main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-security main restricted universe multiverse

# 预发布软件源，不建议启用
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-proposed main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-proposed main restricted universe multiverse

Then, remember synchronizing the package index files after changing the sources.list file: 

sudo apt-get update

1.2 Install packages

You can put a list of packages in a file such as pkg_list.txt: 

vim
tmux
zsh
htop

Then, install them using the following command: 

sudo apt-get install $(grep -vE "^\s*#" pkg_list.txt | tr "\n" " ")

2 Install software from source code

2.1 Emacs

  • Download the source code of Emacs from http://ftp.gnu.org/gnu/emacs/
  • Install the dependencies:
sudo apt-get build-dep emacs24
  • Building Emacs:
cd emacs-25.x
./configure
make
make install
  • Optionally, install Spacemacs:
git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d

2.2 GNU Parallel

wget http://ftp.gnu.org/gnu/parallel/parallel-latest.tar.bz2

2.3 oh-my-zsh

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
chsh -s /usr/local/bin/zsh

2.4 i3wm

downloading: http://i3wm.org/downloads/

3 Install .deb files directly

3.1 Chrome

Download the standalone installation file from: http://www.google.cn/intl/zh-CN/chrome/browser/desktop/index.html

Install: 

sudo dpkg -i google-chrome-stable_current_amd64.deb

4 Install Python modules

4.1 Install pip first

wget https://bootstrap.pypa.io/get-pip.py
sudo python get-pip.py

4.2 Install common modules

Put all modules needed to install in a file named requirements.txt: 

h5py
ipython
pandas
scikit-learn
matplotlib
numpy

sudo -H pip install -r requirements.txt

5 Install software from custom packages

5.1 TexLive

Download texlive.iso, 

wget https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/texlive.iso

Mount texlive.ios and install texlive: 

mkdir ./temp
sudo mount texlive.iso ./temp
sudo ./temp/install.tl

7.1 Keyboard





在 /etc/default/keyboard 文件中添加下面一行，交换 Ctrl 和 CapsLock 键：







XKBOPTIONS="ctrl:nocaps"








完整的 /etc/default/keyboard 文件如下:







XKBMODEL="pc105"





XKBVARIANT=""
XKBOPTIONS="ctrl:nocaps"





BACKSPACE="guess"
