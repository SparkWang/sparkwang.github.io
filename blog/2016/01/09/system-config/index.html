1 Motivation

Debian 系统安装完之后，需要安装很多软件并进行许多配置，这个安装配置的过程应当实现自动化。

2 TODO Method

首先，软件安装有以下几种方式：

  • apt-get 方式安装
  • 从 github 克隆源码后安装
  • 从固定地址下载源码后安装
  • Python pip

需要安装和配置的包括:

  • Window Manager
  • Git
  • sources.list
  • Emacs
  • Keyboard
  • sudo
  • SSH
  • Python
  • Android Development Tools
  • Tmux
  • zsh
  • LaTeX
  • Browser
  • …
